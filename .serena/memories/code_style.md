# Code Style & Conventions

## TypeScript / React
- **Indentation**: 2 spaces
- **Quotes**: Double quotes
- **Naming**:
  - Components: PascalCase
  - Functions/Vars: camelCase
- **Imports**: Explicit imports.
- **Tauri Integration**: Use `@tauri-apps/api/core` for `invoke`.

## Rust
- Standard Rust formatting (`cargo fmt`).
- `snake_case` for functions/modules.
- `PascalCase` for structs/enums.
