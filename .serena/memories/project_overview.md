# Project Overview: CrossEverything

## Purpose
A file search utility for macOS and Linux, mimicking "Everything" on Windows.
Built with Tauri v2.

## Tech Stack
- Frontend: React 19, TypeScript, Vite
- Backend: Rust (Tauri v2)
- Environment: Windows (Development)

## Structure
- `src/`: Frontend source (React components)
- `src-tauri/`: Backend source (Rust)
- `src-tauri/src/lib.rs` (or main.rs): Rust commands
